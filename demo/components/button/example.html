Attribute Description Default
name Text displayed on the button None (required)
primary Applies primary button styling false

light Applies light button styling false
disabled Disables the button and prevents interaction false
submitting Shows loading state during form submission false
small Applies smaller button size false
lined Applies outlined button styling false
