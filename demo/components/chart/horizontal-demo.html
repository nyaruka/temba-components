Attribute Description Default
chartType Type of chart to display ('bar', 'line', 'pie', etc.) 'bar'
horizontal Whether to display the bar chart horizontally false
header Title text displayed above the chart ''
data Chart data object containing labels and datasets None (required)
percent Whether to show percentage values on chart false
maxChartHeight Maximum height for the chart in pixels 250
legend Whether to show the chart legend false

Regular Vertical Bar Chart

Default vertical bar chart display

<temba-chart
  type="bar"
  header="Monthly Sales (Vertical)"
  style="height: 300px; display: block"
></temba-chart>

Horizontal Bar Chart

Bar chart displayed horizontally using the horizontal attribute

<temba-chart
  type="bar"
  horizontal
  header="Monthly Sales (Horizontal)"
  style="height: 300px; display: block"
></temba-chart>

Horizontal Bar Chart with Percentages

Horizontal bar chart showing percentage values

<temba-chart
  type="bar"
  horizontal
  percent
  header="Monthly Sales with Percentages (Horizontal)"
  style="height: 300px; display: block"
></temba-chart>
