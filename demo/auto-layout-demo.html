How to Test Auto Layout:

  1. Drag one of the nodes or sticky notes
  2. Drop it on top of another item to create a collision
  3. Watch as the colliding item automatically moves to make room
  4. Notice how items snap to the 20px grid
  5. Try complex scenarios with multiple overlapping items

Note: This is a visual demonstration of the collision detection logic. The actual implementation is in the Flow Editor component.
