Attribute Description Default
uuid Unique identifier for the sticky note Required
data Object containing position, title, body, and color properties Required
data.position Object with left and top pixel coordinates for positioning { left: 0, top: 0 }
data.title Title text displayed in the header of the note Empty string
data.body Main content text of the sticky note Empty string
data.color Color theme: 'yellow', 'blue', 'pink', 'green', or 'gray' 'yellow'

How to use:
  • Click and drag the sticky notes to move them around
  • Click on the title or body text to edit it inline
  • Press Enter or Escape to finish editing
  • Notes snap to a 20px grid when you finish dragging
  • Each color has its own visual theme
