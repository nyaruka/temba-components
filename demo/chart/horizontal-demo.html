TembaChart Example

← Back to main demo

Attribute Description Default
type Chart type: 'bar', 'line', 'pie', 'doughnut' 'bar'
horizontal Boolean to render horizontal bar charts false
header Title text displayed above the chart Empty string
data Chart data object with labels and datasets arrays Required
percent Boolean to show percentage labels on the chart false
url URL endpoint to fetch chart data undefined
palette Color palette key for chart colors 'qualitative-set1'
legend Boolean to show chart legend false

Basic Vertical Bar Chart

A standard vertical bar chart with multiple datasets

<temba-chart
  type="bar"
  header="Monthly Sales (Vertical)"
  style="height: 300px; display: block"
></temba-chart>

Horizontal Bar Chart

The same data displayed as a horizontal bar chart using the horizontal attribute

<temba-chart
  type="bar"
  horizontal
  header="Monthly Sales (Horizontal)"
  style="height: 300px; display: block"
></temba-chart>

Horizontal Chart with Percentages

A horizontal bar chart that displays percentage labels on each bar

<temba-chart
  type="bar"
  horizontal
  percent
  header="Monthly Sales with Percentages"
  style="height: 300px; display: block"
></temba-chart>

Chart with Legend

A horizontal chart displaying a legend to identify different data series

<temba-chart
  type="bar"
  horizontal
  legend
  header="Monthly Sales with Legend"
  style="height: 350px; display: block"
></temba-chart>
